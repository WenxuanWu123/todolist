# Todo List 桌面应用

一个简单易用的桌面Todo List应用，支持窗口拖动、固定和调整大小功能，以及亮色/暗色主题切换。

## 功能特点

- ✅ 可拖动的主窗口
- ✅ 可拖动的编辑窗口
- ✅ 窗口固定/取消固定功能
- ✅ 窗口左右调整大小
- ✅ 亮色/暗色主题切换
- ✅ 日历选择器设置截止日期
- ✅ 添加、编辑、删除任务
- ✅ 标记任务完成状态
- ✅ 一键删除已完成任务
- ✅ 数据自动保存

## 使用方法

### 启动应用

#### 方法1：直接运行Python脚本

```bash
python todo_app.py
```

#### 方法2：使用VBS脚本（无控制台窗口）

```bash
cscript run_todo.vbs
```

### 添加任务

1. 在「任务」输入框中输入任务描述
2. 点击「📅」按钮选择截止日期
3. 点击「添加任务」按钮

### 编辑任务

1. 选择要编辑的任务
2. 点击「编辑任务」按钮
3. 在弹出的对话框中修改任务信息
4. 点击「保存」保存修改

### 删除任务

#### 单个删除
1. 选择要删除的任务
2. 点击「删除任务」按钮
3. 在确认对话框中点击「是」

#### 批量删除已完成任务
1. 点击「删除已完成任务」按钮
2. 在确认对话框中点击「是」

### 标记任务状态

1. 选择要标记的任务
2. 点击「切换状态」按钮
3. 任务状态会在完成（✓）和未完成（✗）之间切换

### 窗口操作

- **拖动窗口**：点击窗口任意位置并拖动
- **调整窗口大小**：将鼠标移动到窗口左右边缘，当光标变为双向箭头时拖动
- **固定窗口**：点击右上角的📌按钮，勾选表示窗口置顶
- **切换主题**：点击右上角的🌙按钮，切换亮色/暗色模式

## 数据存储

任务数据和主题偏好会自动保存到用户主目录下的 `.todo` 文件夹中：

- `~/.todo/todos.json` - 任务数据文件
- `~/.todo/theme.json` - 主题偏好设置

## 开发说明

### 主要文件结构

- `todo_app.py` - 主应用程序文件
- `run_todo.vbs` - 无控制台启动脚本
- `README.md` - 说明文档
- `~/.todo/todos.json` - 任务数据文件（自动生成）
- `~/.todo/theme.json` - 主题偏好设置（自动生成）

### 核心功能模块

1. **窗口管理** - 窗口拖动、调整大小、固定功能
2. **主题管理** - 亮色/暗色主题切换、主题偏好保存
3. **任务管理** - 添加、编辑、删除、状态切换
4. **日期选择** - 弹出式日历选择器
5. **数据持久化** - JSON格式数据存储
